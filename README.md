# 🌍 Africa Team Telegram Bot (@AfricaTeamBot)

AI-powered Telegram coaching bot for all Africa Team products.

## Products Covered
- 🔮 Coinverse — Prediction markets, 100 USDW free
- 🤖 UTrading AI — AI trading 24/7, code AFRICA  
- ⚡ Eropia DeFi 4.0 — Stake USDC up to 12X
- 💝 KindFlow — P2P crowdfunding from $25

## Commands
/start /products /coinverse /utrading /eropia /kindflow /coach /signals /help

## Features
- Keyword routing (COINVERSE, UTRADING, EROPIA, KINDFLOW, NFT, STAKING...)
- FAQ auto-response (5+ answers per product)
- AI Claude coaching for unknown questions
- Group welcome messages
- Weekly performance report to Coach
- Inline keyboard buttons

## Deploy on Railway
1. Connect this repo in Railway dashboard
2. Set env vars (see .env.example)
3. Deploy — bot goes live 24/7
